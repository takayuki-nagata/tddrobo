# TDDRobo

TDDRobo is an autonomous AI agent workflow powered by **LangGraph** and **Google GenAI** that builds software using a strict **Test-Driven Development (TDD)** approach.

Given a high-level goal and a specification URL, the agent will:
1. Read and parse the specification.
2. Generate a system design document.
3. Plan and write a comprehensive `pytest` suite (Red phase).
4. Implement the logic to make the tests pass (Green phase).
5. Automatically debug, fix syntax errors, and iterate until all tests succeed (Refactor phase).
6. Generate a final `README.md` for the generated project.

## ⚠️ Security Warning

**This agent automatically executes Python code generated by an LLM on your local machine.** 
It uses `subprocess` to run syntax checks and `pytest`. While it is designed to build standard software, AI-generated code can be unpredictable. 
**It is highly recommended to run this project inside an isolated environment**, such as a Docker container, DevContainer, or a secure Virtual Machine.

## Prerequisites

- Python 3.14+
- uv (Fast Python package installer and resolver)
- A Google Gemini API Key (`GEMINI_API_KEY`)

## Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/tddrobo.git
   cd tddrobo
   ```

2. Install dependencies using `uv`:
   ```bash
   uv sync
   ```

3. Configure your API key:
   Create a `.env` file in the project root and add your Gemini API key:
   ```env
   GEMINI_API_KEY=your_api_key_here
   ```

## Usage

You can start the workflow by running `cli.py` with your goal and specification URL:

```bash
uv run python cli.py \
  --goal "Build a simple calculator" \
  --spec-url "https://example.com/calculator-spec"
```

Alternatively, you can run the provided demonstration script which attempts to build a POSIX-compliant `bc` clone:

```bash
uv run python run_bc_demo.py
```

### Artifacts
All generated code, test plans, design documents, and SQLite databases for MLflow tracking are saved in the `artifacts/` directory by default.

## Development & Testing

This project enforces strict code quality checks using `Ruff`, `Mypy`, and `Pytest` with 100% test coverage.

To run the CI checks locally:
```bash
chmod +x tests/run_tests.sh
./tests/run_tests.sh
```

## MLflow Tracking
The agent uses MLflow to trace LLM calls and log workflow metrics. You can view the traces by starting the MLflow UI:
```bash
uv run mlflow ui
```
